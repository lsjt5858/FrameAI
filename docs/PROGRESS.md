# FrameAI 开发进度

更新时间：2026-05-24

这个文档用于跟踪 FrameAI 的开发进度。每完成一个模块，就在对应 checklist 中打勾，并在“完成记录”里补一条简短记录。

## 状态说明

- `[ ]`：未开始
- `[-]`：进行中
- `[x]`：已完成
- `[~]`：已完成基础版，后续需要增强

## 当前阶段

当前处于 `Phase 1: MVP 闭环`。

第一阶段目标：跑通“创建项目 -> 上传素材 -> 创建分镜 -> 套用提示词 -> 创建生图任务 -> 保存结果 -> 创建生视频任务 -> 保存结果 -> 标记入选”的最小闭环。

## 更新规则

每完成一个模块时，按下面顺序更新：

1. 把对应 checklist 从 `[ ]` 改成 `[x]`，如果只是基础版先改成 `[~]`。
2. 如果整个小模块都完成，同步更新阶段总览状态。
3. 在“完成记录”新增一行，写清楚日期、模块和结果。
4. 如果发现新的风险或遗留问题，补到“技术债与注意事项”。
